# Real-Time Collaborative Text Editor

## 🚀 Features
- Real-time text editing using Socket.io
- Multiple users can edit simultaneously
- Live cursor tracking
- User count display

## 🛠 Tech Stack
- Node.js
- Express
- Socket.io
- HTML, CSS, JavaScript

## 🧠 Architecture
Client ↔ Socket.io ↔ Server (Node.js)

## ⚠️ Limitations
- Data is not stored permanently (no database)

## 🔮 Future Improvements
- Add MongoDB for persistence
- User authentication
- Shareable document links

## 🤖 AI Tools Used
- ChatGPT (for guidance and debugging)

## ▶️ Setup Instructions
1. npm install
2. node server.js
3. Open http://localhost:3000
